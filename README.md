# The Lav Forge - Elite WoW Studio Web Application

Welcome to the official web application for **The Lav Forge**, an elite World of Warcraft development studio specializing in custom Addons, complex WeakAuras, and high-performance user interface overhauls.

This web application is built using a modern, fast, and highly-optimized tech stack, designed to deliver a visually stunning, responsive, and fully localized experience.

## 🚀 Technology Stack

- **Core Framework:** [Next.js](https://nextjs.org/) (App Router, localized routing)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) & Vanilla CSS with beautiful fluid grids and glassmorphism.
- **Animations:** [Framer Motion](https://www.framer.com/motion/) for fluid page transitions, micro-interactions, and premium UI feedback.
- **Localization:** Fully integrated multi-language support (English, German, French, Spanish, Russian, Chinese) with dynamic translation lookup.
- **Performance & SEO:** Native Next.js sitemaps, structured schema markup (JSON-LD FAQ data), and Google Analytics 4 integration.

## 🛠️ Getting Started

To run the development server locally, follow these steps:

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

4. **Start production server:**
   ```bash
   npm run start
   ```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

## 📁 Directory Structure

- `src/app` - Contains localization-wrapped Next.js page routes, layout setups, and API protocols.
- `src/components` - Reusable UI elements, canvas effects (e.g., interactive fire particles, ocean backgrounds), and feature sections.
- `src/dictionaries` - High-fidelity localization translation maps in JSON.
- `src/lib` - Multi-language support configuration, search engine optimization utilities, and helper scripts.
